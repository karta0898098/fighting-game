# 音效檔（SFX）

把音效檔依「事件名稱」放在此目錄即可生效，**程式會自動依檔名載入，不必改任何程式碼**。
缺檔時該音效靜音、不會報錯（可逐步補齊）。

> 目前已附上**合成的佔位音效**（`.wav`）供試玩；要換成正式音效，只要放上同名的 `.mp3`（會優先採用）即可，舊的 `.wav` 佔位檔可刪可留。

## 命名慣例

檔名 = 事件名，副檔名 `.mp3`（正式，優先）或 `.wav`（佔位）；同名兩者都在時以 `.mp3` 為主。例如 `swing.mp3`。

| 檔名         | 觸發時機                             | 目前狀態      |
| ------------ | ------------------------------------ | ------------- |
| `footstep1` `footstep2` `footstep3` | 走路腳步（按一下一聲、按住連續；每步隨機選一） | 🎵 wav 佔位     |
| `swing`      | 近戰／衝刺類出手（普攻揮砍）             | 🎵 wav 佔位     |
| `cast`       | 施法／遠程／增益類出手                 | 🎵 wav 佔位     |
| `dash`       | 衝刺技                                 | 🎵 wav 佔位     |
| `blink`      | 瞬移／閃現技                           | 🎵 wav 佔位     |
| `ultimate`   | 大招施放                                 | 🎵 wav 佔位     |
| `hit`        | 攻擊命中／撞擊                           | 🎵 wav 佔位     |
| `hurt`       | 角色受傷（HP 下降）                      | 🎵 wav 佔位     |
| `death`      | 角色死亡                                 | 🎵 wav 佔位     |
| `buff`       | 增益／護盾（如需與 cast 區分時）        | 🎵 wav 佔位     |

## 每角色專屬音效（可選，會覆寫泛型）

以該動作的 **vfx id** 命名即可覆寫上表的泛型音效；缺檔時自動回退泛型名。
範例（同樣可用 `.mp3` 或 `.wav`）：

- `warrior_grapple` → 戰士「鉤索」專屬出手音（缺檔則用 `swing`）
- `mage_fireball`    → 法師火球專屬施法音（缺檔則用 `cast`）
- `fighter_ultimate` → 格鬥家大招專屬音（缺檔則用 `ultimate`）

vfx id 對應請見 `src/game/characters.js` 各角色 `basic/skill1/skill2/ultimate` 的 `vfx` 欄位。

## 檔案格式

- 載入順序：先找 `.mp3`（正式、相容性最佳），再找 `.wav`（佔位）；以能成功解碼的為主。
- 短音效請保持精簡（出手／命中通常 < 1 秒）；腳步請放「單一一步」的短音效（footstep1/2/3），程式會每步隨機選一 + 微調音高/音量。
- 附上的佔位音效為程式合成（純振盪器 + 包絡），僅供佔位試玩、請自行換成正式素材。

## 行為說明

- **空間化**：音效依與本地玩家的距離自動衰減音量，並做左右聲道定位（遠處角色較小聲）。
- **觸發分工**：出手／施法／大招／受傷／腳步由渲染端依角色狀態觸發；命中／死亡由特效事件觸發（不會重複發聲）。
- **音樂**：背景音樂在 `../music/`，與此處音效系統各自獨立。
